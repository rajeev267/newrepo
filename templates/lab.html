Our Department Laboratories

C Programming Lab

Purpose: Learn programming fundamentals and problem-solving using C.

Students Learn:

  • Loops, conditionals, functions
  • Arrays, strings, and pointers
  • File handling

Tools Used: GCC / Code::Blocks

C++ Programming Lab

Purpose: Understand Object-Oriented Programming principles using C++.

Students Learn:

  • Classes, objects, and inheritance
  • Polymorphism and encapsulation
  • File handling and templates

Tools Used: Code::Blocks / Dev C++

Java Programming Lab

Purpose: Build console and GUI-based programs using Java.

Students Learn:

  • OOP concepts and exception handling
  • GUI using Swing/JavaFX
  • Database connectivity (JDBC)

Tools Used: NetBeans / IntelliJ IDEA

Web Development Lab

Purpose: Learn to design and build interactive websites.

Students Learn:

  • HTML for structure
  • CSS for styling
  • JavaScript for interactivity

Tools Used: VS Code / Browser Dev Tools

.NET Programming Lab

Purpose: Learn to develop Windows and web applications using .NET.

Students Learn:

  • Basics of C# programming
  • Form and web-based applications
  • Database connectivity via ADO.NET

Tools Used: Visual Studio

Android Development Lab

Purpose: Create mobile applications for Android.

Students Learn:

  • UI design and layout creation
  • Activity lifecycle
  • Database and API integration

Tools Used: Android Studio

RDBMS Lab

Purpose: Understand relational database systems and SQL.

Students Learn:

  • Database creation and management
  • SQL queries (SELECT, UPDATE, etc.)
  • ER diagrams and normalization

Tools Used: MySQL / Oracle

IoT (Internet of Things) Lab

Purpose: To provide hands-on experience in building smart systems that connect hardware devices with software and the internet.

Students Learn:

  • Working with microcontrollers (Arduino, NodeMCU, ESP32)
  • Sensor and actuator interfacing
  • Data transmission using Wi-Fi, Bluetooth, or MQTT
  • IoT-based project development (smart home, automation, etc.)

Tools Used: Arduino IDE / Node-RED / Blynk / MQTT Broker
